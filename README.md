# SkillQ
A web based Skill Monitor for Eve Online.

[![Login with EVE Online](https://skillq.net/img/ssologin.png)](https://skillq.net/login-check)

### What is SkillQ?
SkillQ allows you to monitor your skills in a single place using any browser..
